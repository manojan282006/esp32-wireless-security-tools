## 📖 Overview

ESP32 Wireless Security Tools is a collection of embedded security projects developed using the ESP32 microcontroller. This repository demonstrates different wireless security concepts through practical implementations, including OLED-based interfaces and Web-based interfaces.

The primary objective of this repository is to explore ESP32 capabilities in wireless networking, embedded systems, and cybersecurity while providing educational resources for students, researchers, and developers.

---

## ✨ Features

- ESP32 based implementations
- OLED Display Interface
- Web Dashboard Interface
- Wireless networking experiments
- Embedded system development
- Hardware integration
